Предприятие

Создать приложение для учета сотрудников предприятия. Оно должно отобразить список всех сотрдников. Предоставить форму для ввода нового сотрудника. Добавить функционал для сортировки списка сотрудников по разным ключам, задать поиск сотрудника по разным ключам. Каждый сотрудник имеет: уникальный номер; имя; дату поступления на работу; дату рождения; электронный адрес. Даты определяются только годом (год поступления на работу и год рождения).
